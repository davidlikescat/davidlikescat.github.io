Section 03 · 과정에 앞서

Attitude

6 Mindsets

실습을 시작하기 전,
6가지 마음가짐.

01

처음엔 벽이 있어요.
당연합니다.

벽은 두 종류입니다.

🧱
환경의 벽
코드베이스 · IDE · 용어 · 폴더 구조 — 도구 자체가 처음인 낯섦.
🧩
빌드의 벽
"폴더 어디 저장된 거지? 어떻게 동작하는 거지? 제대로 만든 거 맞나?"

강사도 그랬습니다. 지금은 자연스럽습니다.

02

반복하면
누구나 됩니다.

한 번, 두 번, 세 번, 네 번.
의구심은 만들수록 사라집니다.

강사도 처음엔 똑같이 막막했습니다.

그리고 Claude에게 물어보면, 너무나 친절하게 알려줍니다.
03

단순한 것부터,
어려운 것은 나중에.

작은 에이전트 하나 → 동작 확인 → 또 만들기.
"내 업무에 적용할 게 뭔지" 자연스럽게 보입니다.

04

개인적 페인 포인트가
가장 좋은 출발점입니다.

업무 외에도 페인 포인트는 분명히 있습니다.
호기심이 동력입니다.

청약홈 · 유튜브 요약 · 캘린더 자동 등록 — 모두 제 개인 페인에서 출발했습니다.

05

많이 보고,
많이 만들어 보세요.

짤막한 워크플로우도 경험치.
경험이 쌓이면 에이전트들이 연결됩니다.

디스코드→ Claude→ 노션→ GitHub Pages

강사 80개 폴더 = 양적 축적의 산물

06

겁먹지 말고,
일단 하세요.

주말 · 저녁 30분~1시간이면 충분합니다.

그냥 해. 그냥 하는 거야.
그냥 해. 그냥 해!

네이버 클립 + 쓰레드 4개월 1억 2천

48% 수수료가 만든 새로운 블루오션 — 수익화 관점 + 비판적 분석

2026-05-28 원본 영상 원본 노션
TL;DR

기회는 진짜다. 단, 영상에서 말하는 "48%"와 "블루오션"이라는 단어는 다소 과장이다. 진짜 차익거래는 수수료율이 아니라 네이버가 자체 트래픽을 갈아넣는 짧은 골든 윈도우에 있다.

1. 사실관계 검증 — 영상 vs 공식자료

영상 주장공식 자료 (2026-05 기준)판정
수수료 최대 48%상품별 5~28% 차등 + 이벤트 3% 보너스, 프리미엄 최대 15%, 간접 1.8%과장
쿠팡 파트너스 3%카테고리별 1~3% (가전 1%, 뷰티 3%)사실
유튜브 쇼핑 6%카테고리별 다름대체로 사실
"거의 아무도 모름"2025-07-23 정식 출시, 2026-02 크리에이터 2만명 확대레드오션화 중
"계정 삭제 거의 없음"저작권 경고 1회 안내 → 2회 30일 정지 → 3회 영구정지조건부 사실

→ 48%는 셀러가 자율 설정한 극단적 프로모션 한정 케이스거나, 1회성 캠페인이었을 가능성. 일반 상품 기대값은 5~15% 수준으로 보는 게 현실적.

2. 수익화 관점 — 따라하려면 이렇게

2.1 셋업 (반나절)

  1. 네이버 브랜드 커넥트 → 크리에이터 스페이스 개설 (네이버 계정만 있으면 가능, 사전심사 없음)
  2. 본인 채널 1개 이상 등록 (블로그/포스트/SNS 어느 거나 가능)
  3. 계정 3개까지 가능 → 본인 + 가족 명의로 분산 시 트래픽·실험 병렬화
  4. 정산 계좌 등록, 최소 정산금 1만원

2.2 콘텐츠 파이프라인 (1편당 30분~1시간 목표)

[1] 영상 소스 줍기 (Threads)
    └─ 도구: Chrono Sniffer (Chrome 무료 확장)
    └─ 카테고리: 뷰티/필링기/아쿠아 시술 (시각 후킹 강한 것)

[2] 제품 식별
    └─ 캡처 → ChatGPT/Gemini "이 제품 카테고리?"
    └─ 답변 키워드 → 네이버 브랜드 커넥트 검색

[3] 상품 선정 (2가지만 본다)
    └─ 가격대: 3~7만원 (충동구매 골든존)
    └─ 리뷰 수: 0이면 무조건 제외 (전환율 0)

[4] 영상 가공
    └─ 클립용 정사각형 템플릿 (상하단 잘림 미리 계산)
    └─ 후킹 카피: 상단 대형, 형광초록/핫핑크
    └─ 길이: 짧을수록 좋음 (쇼츠 20~30초 X, 7~12초 권장)

[5] 카피 생성
    └─ GPT/잼스로 후킹 10개 뽑고 1개 선택
    └─ 패턴: 숫자+베네핏 / 도발형 / 타깃 리액션형 / 해시태그형

[6] 업로드 (네이버 클립은 모바일 전용)
    └─ 본문에 검색 키워드 박기 (네이버 알고리즘 핵심)
    └─ 광고 표시 문구는 본문 최상단

[7] 동시 발행 (쓰레드)
    └─ 톤만 MZ로 바꿔서 같은 영상 재활용
    └─ 영상 + 이미지 1장 같이 (링크 가림 방지)
    └─ 단축링크 (인포크/머닝) 처리
    └─ 광고 문구는 링크 위에

2.3 타겟 세그먼트 매트릭스

플랫폼타깃충동구매 가격대잘 팔리는 카테고리후킹 코드
네이버 클립시니어 여성 40~60대3~7만원미백·모공·립·필링기"5살 어려보임", "처짐 사라짐"
쓰레드MZ 20~30대1~5만원뷰티·라이프·잇템"야르", "은근 힐링됨" 신조어 필수

2.4 수익 시뮬레이션 (현실 보정)

야미 케이스에서 영상 수수료율 48%를 현실 평균 12%(5%+이벤트+프리미엄 평균)로 보정하면:

항목영상 주장보정 추정
4개월 누적 매출12.6억12.6억 (GMV 그대로)
4개월 수익1.16억 (≈9.2%)0.6~1.5억 (5~12% 범위)
월 평균 수익2,900만원1,500~3,700만원

→ 수익은 여전히 큼. 단, "48% 수수료 → 매출의 절반이 수익"이라는 그림은 환상.

3. 비판적 시각 — 7가지 리스크/함정

3.1 「48% 수수료」는 마케팅 미끼

공식 평균은 5~28%, 흔히는 8~15%. 48%를 메인 카피로 쓴 시점에서 이 영상은 야미 본인의 강의/멘토링 유입 채널일 가능성이 높다. ("꽃수이/민쌤/주투/루드 멘토링 성공 사례"가 줄줄이 나옴 = 강의 판매 시그널)

3.2 영상 도용 = 시한폭탄

  • "쓰레드 해외 계정에서 줍는다"는 무단 복제·전송권 침해 (저작권법 제16조, 제18조)
  • 해외 저작권자가 직접 신고 안 해도, 한국저작권보호원이 모니터링 + 직권조사 가능
  • 네이버 클립 정책: 2회 = 30일 수익정지 / 3회 = 영구정지
  • 누적 수익 1억+ 단계에서 영구정지 시 백업 채널 없으면 즉시 0원

3.3 식약처 광고 규제 직격탄

미백·모공·주름 = 기능성 화장품 카테고리

  • "5살 어려보임", "각질 제거 하나로 인상 5살 어려짐" → 부당 의료 효과 표시·광고 가능성 (화장품법 제13조)
  • 시니어 타깃에 자극적 카피일수록 식약처/공정위 모니터링 대상
  • 위반 시: 시정명령 → 과징금 → 영업정지 (개인 셀러도 동일)

3.4 광고 표시 규제 — 작은 글씨 함정

  • "광고: 이 게시물은 쿠팡 파트너스..." 문구를 링크 위에 적어야 한다는 영상 팁은 정확
  • 단, 공정위 「추천·보증 등에 관한 표시·광고 심사지침」은 글자 크기/색상 대비도 명시. 작은 회색 글씨로 흘려 적으면 처벌 대상
  • 신고 포상금제 활성 (1건 최대 200만원) → 동종 부업자가 경쟁자 신고하는 케이스 빈번

3.5 시장은 이미 정원초과

  • 2026-02 기준 네이버가 클립 크리에이터 2만명 확대 모집 발표
  • 영상이 "거의 아무도 모름"이라 한 시점 대비 진입자 폭증 중
  • 같은 쓰레드 소스를 너도나도 쓰면 → 영상 중복도 패널티 (네이버 알고리즘이 이미 학습 중)
  • 6~12개월 후엔 쿠팡 파트너스처럼 평균 수익 추락 가능성 높음

3.6 "1억 2천"의 분모 문제

  • 누적 매출 12.6억 ÷ 두 계정 × 4개월 = 월 1.57억 GMV
  • 평균 객단가 5만원 가정 시 월 3,000건 판매 = 일 100건
  • 이걸 만들려면 클립 영상 일 5~10편 + 쓰레드 동시 발행 = 외주·알바 인건비 발생
  • 영상 외주 1편당 1~3만원 → 월 인건비 300~900만원
  • "1인이 부업으로 시작해서" 도달 가능한 수치 아님. 야미 본인도 "프리랜서 외주 + 알바 업로드" 시스템화했다고 명시

3.7 플랫폼 리스크 100% 네이버 의존

  • 네이버가 수수료 정책을 언제든 변경 가능 (2025-08 간접수수료 1.8% 추가가 그 예)
  • 쿠팡 파트너스도 초기 8% → 3% 단계적 인하 전례
  • 네이버는 트래픽 키우려고 지금 공격적 보너스 푸는 단계 → 6~12개월 내 수수료 표준화(낮춤) 가능성 매우 높음
  • 야미 자신도 "지금이 골든타임"이라 명시한 이유

4. 누구에게 추천하나

조건추천도
본업 있는 1인개발자, 시간 주당 5시간 미만비추천 "쉬운 부업" 아님. 시스템화 전엔 시간 블랙홀
이미 시니어 뷰티 도메인 지식/네트워크 있음추천 컨텐츠 차별화 가능
단기 6개월 안에 1~3천만원 캐시 필요조건부 도용 리스크 받아들이고 한탕
합법적·지속적 수익 채널비추천 본인 사진/리뷰 + 정식 PPL이 안전
AI 워크플로우 학습용추천 콘텐츠 파이프라인 학습 케이스로 좋음

5. 만약 한다면 — 리스크 최소화 체크리스트

  1. 소스 영상은 본인 촬영 or 제휴 권한 받은 것만 (해외 도용 X)
  2. 광고 문구는 굵게·검정·14pt 이상, 본문 최상단
  3. 미백/주름/항노화 표현 금지 → "관리감", "보습", "톤업" 같은 안전 표현
  4. 계정 3개 분산 + 백업 채널 (인스타 릴스 등 동시 운영)
  5. 수익률 가정은 8~12%로 잡고 BEP 계산
  6. 분기마다 네이버 정책 변경 모니터링 (수수료율 표 캡처 보관)
  7. 3개월 누적 수익이 인건비 안 넘으면 손절 기준 미리 정해둘 것
